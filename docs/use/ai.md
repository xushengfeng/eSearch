# 在线 AI 模型

用于编辑 OCR 校对、AI 以图搜图、翻译、在线 OCR 等

## 本地 LLM

你可以使用任何技术栈部署 LLM，但需要兼容 OpenAI 的 API。下面以 ollama 为例。

你需要在[ollama 官网](https://ollama.com/)下载并安装 ollama，添加模型。

在在线AI中添加，填入 url：`http://localhost:11434/`，点击获取模型列表，选择你想要的模型。确保测试通过。
